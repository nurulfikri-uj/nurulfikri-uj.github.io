Kuisioner : Manajemen Diri

1. Saya merencanakan agenda kegiatan sehari-hari

2. Ketika melakukan pekerjaan, saya sering kesulitan karena kehilangan benda yang diperlukan

3. Saya mencatat aktivitas harian

4. Saya sering terlambat ke sebuah acara

5. Jika akan mengunjungi seseorang, saya menelponnya terlebih dahulu

6. Saya melakukan aktivitas tanpa direncanakan

7. Saya membawa buku agenda/notes dan pulpen kemanapun saya pergi

8. Saya sering mengobrol dan bercanda bersama teman-teman

9. Saya sholat tepat pada waktunya

10. Saya sering wa-an dengan teman-teman walaupun tidak ada berita penting

11. Ketika di mobil/pesawat/bus/kereta saya terbiasa membaca

12. Kadangkala saya menelpon teman hanya untuk mengobrol

13. Ketika membuat janji, saya memastikan kedua belah pihak mengerti waktunya dengan jelas

14. Ketika main game, saya sering lupa waktu

15. Saya menepati/melakukan kegiatan yang sudah direncanakan

16. Saya suka browsing hingga larut malam

17. Untuk suatu keperluan, dapat diselesaikan dengan mengirim email/wa

18. Seminggu sekali saya jalan-jalan ke mall

19. Saya menyediakan uang kecil untuk parkir dan keperluan lainnya

20. Saya melayani percakapan telpon dari teman walau saya sadar yang dibicarakan tidak/kurang penting

21. Ketika hendak berbelanja, saya mencatat barang yang akan dibeli

22. Kadang saya lupa mengisi tangki kendaraan hingga kehabisan bensin

23. Jika akan pergi ke suatu tempat, saya mencari rute tercepat untuk mencapainya

24. Kadang saya harus mencari tukaran uang receh untuk membayar sesuatu
