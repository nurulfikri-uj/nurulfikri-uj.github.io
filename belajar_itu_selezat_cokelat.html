Kuisioner : Belajar itu Selezat Cokelat

1. Saya baru akan belajar dirumah jika akan ulangan dan mengerjakan PR

2. Saya selalu bersemangat jika akan berangkat ke sekolah

3. Saya kurang antusias dengan pelajaran yang gurunya membosankan dalam mengajar

4. Saya selalu antusias dan berusaha mencoba dengan semua mata pelajaran

5. Saya merasa tenang jika teman-teman mendapatkan nilai rendah sama seperti nilai yang saya peroleh

6. Saya selalu mencoba latihan soal-soal yang menantang walau tidak disuruh

7. Saya selalu pusing dan bosan melihat buku-buku pelajaran terutama pelajaran eksak

8. Saya sudah membuat/ memiliki jadwal khusus untuk belajar setiap malam

9. Saya selalu menunda-nunda untuk mulai belajar sampai akhirnya tidak jadi belajar

10. Saya selalu bersemangat belajar karena saya tahu manfaatnya untuk diri saya dimasa depan

11. Saya senang jika guru tidak hadir

12. Saya senang jika berkumpul dengan teman-teman untuk diskusi pelajaran

13. Saya selalu membicarakan guru “killer” yang membosankan dalam mengajar dengan teman-teman

14. Saya sering membantu teman yang kesulitan dalam mata pelajaran tertentu

15. Nilai saya selalu jelek jika diajar oleh guru “killer” yang kurang enak dalam mengajar

16. Saya sering mendapat nilai bagus dimata pelajaran yang dianggap teman-teman gurunya “killer”

17. Saya takut jika guru mengadakan ulangan dadakan

18. Saya sering diandalkan oleh teman-teman untuk menjelaskan ulang pelajaran yang telah di ajarkan oleh guru

19. Saya lebih sering mengerjakan PR di sekolah saat akan masuk kelas dengan melihat pekerjaan teman.

20. Saya sering menghabiskan waktu dengan belajar atau membaca
