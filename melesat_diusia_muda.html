Kuisioner : Melesat Diusia Muda

1. Ketika ada waktu luang saya mengisinya dengan menyalurkan hobi

2. Saya merasa hidup saya selalu bermasalah

3. Setiap pagi saya selalu bangun lebih awal

4. Saya senang nongkrong dengan teman-teman untuk menghabiskan waktu

5. Saya selalu dipercaya untuk menjadi pemimpin oleh teman-teman saya

6. Jika sedang ada masalah saya sering mengeluhkanya

7. Ketika ujian saya pantang untuk mencontek atau bertanya

8. Saya senang ikut-ikutan trend apa saja

9. Saya selalu bertanya jika ada pelajaran yang saya tidak pahami

10. Saya kesal jika teman saya tidak membantu saya saat sedang ujian

11. Saya sering mendapat nilai tinggi di sekolah

12. Saya susah bangun pagi

13. Jika ada teman yang menyinggung atau menghina saya, saya tidak pedulikan

14. Saya selalu menghindar jika ada kesempatan bertanya kepada guru

15. Jika ada keinginan saya berusaha mengkomunikasikan dengan orangtua

16. Saya benci sama orang yang menyinggung perasaan saya

17. Saya sudah menulis impian dan cita-cita yang saya inginkan

18. Saya sering memaksakan kehendak walaupun orang tua tidak setuju

19. Saya senang menolong teman yang sedang kesulitan

20. Saya belum tahu dan belum punya rencana nanti saya mau jadi apa
